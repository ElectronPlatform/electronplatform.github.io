Tester Notes

Before testing

Electron Preview is intended for early testers. It may change quickly, and feedback packages are created locally so testers can review what they share.

Preview Notes

What to test first

  • Open the app and confirm the welcome flow feels clear.
  • Connect a supported RFID device and run safe read-only scans.
  • Create a feedback package when something behaves unexpectedly.
  • Do not share private keys, dumps or personal card data unless you choose to include them.

Changelog

Current focus

The first public Preview focuses on Device Console stability, Card Intelligence, Report generation, local feedback packages, Preview status and a safer testing workflow.
