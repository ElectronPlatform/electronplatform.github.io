Features

Electron modules

Electron is built from connected parts: knowledge, intelligence, reports, device safety, device management and feedback.

Platform modules

Knowledge Engine

Understand what Electron sees

Card families, protocols and safe next steps are explained in plain language so the user does not need to start with command-line knowledge.

Card Intelligence

Turn scans into meaning

Electron combines scan output, confidence, readable data and research context into a clearer card profile.

Report Engine

One model, multiple reports

Reports are built from a central Report Model so TXT, DOC, PDF and future exports can share the same source of truth.

Device Guard

Predictable hardware workflows

Device checks and command safety rules help prevent confusing hangs, unsafe commands and workflows that start without a suitable reader.

Device Manager

Prepared for multiple readers

Electron is moving toward device profiles, capabilities and command libraries instead of hardcoding one reader everywhere.

Feedback Engine

User-controlled support packages

Feedback packages can include notes, screenshots, logs and console output, but nothing is sent automatically.
