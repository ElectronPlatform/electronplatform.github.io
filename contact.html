Feedback packages

User controlled

The Electron app can create a local feedback package with selected text, screenshots, logs and device console output. Review the package before sharing it.

Before sharing

Privacy reminder

  • Do not share private keys, card dumps or personal card data unless you intentionally choose to include them.
  • Feedback ZIP files are created locally first, so you can inspect them before sending.
  • Google Drive is used only for release downloads. Support messages are handled through the official email address for now.
